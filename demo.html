css-toolset.css

This is a small set of tools that aims to generate a starter framework from which you can start generating mobile sites with more modern styles.

To do this, several styles are overwritten by default browsers.

Status: In development (not for production)

Tools and Utilities

  • .flat - Remove the rounded edges.
  • .Sol - Element with color background I define in the variable --hover.
  • .Com - Swap colors for their complements.
  • .Out - Remove the border from the elements
  • .Inv - Inverts the colors defined in the variables --text and --hover.
  • hr.Sep - Invisible 1rem separation (only for hr tags)
