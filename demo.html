css-toolset.css

This is a small CSS stylesheet that provides a set of enhancements and utility classes that make your website look more elegant, usable and responsive, mainly on small screens.

You just need to import the stylesheet into your document and all the default styles will be applied.

Status: In development (not for production)

Utility Classes Available

  • .flat - Remove the rounded borders.
  • .Sol - Element with color background defined by css variable --hover.
  • .Com - Swap the colors between they complementary.
  • .Out - Remove the border.
  • Inv - Inverts the colors defined in the variables --text and --hover.
  • hr.Sep - Invisible 1rem separation (only valid for hr tags)
